Cursor Ring & Magnifier
ストーリー デモ 画面 入手

講師業から生まれた、実戦用macOSアプリ

伝わる解説は、
カーソルから始まる。

オンライン講座でも、対面講座でも、 「どこを見ればいいか」が一瞬で伝わる。 Cursor Ring & Magnifier は、講師本人が現場で使うために作った カーソル可視化ツールです。

リング強調 クリック可視化 拡大鏡 日本語設定UI
最新版をダウンロード GitHubを見る

なぜ作ったのか

制作者自身が講師業をしている中で、 「マウスカーソルの位置が受講者に伝わらない」という課題を 何度も感じたことが開発のきっかけです。

目指したもの

派手さよりも、説明のしやすさ。 講義中に迷わない設定画面、必要な機能へすぐ届く操作性、 実運用で気持ちよく使えることを重視しています。

アプリ操作デモ

実際の動きが一番早いです。リング・クリック・拡大鏡の流れを動画で確認できます。

設定画面と操作イメージ

`settei1~3` / `sousa1~4` の画像を配置すると、自動でギャラリー表示されます。

今すぐ使う

GitHub Releases から CursorRingMagnifier-macOS.zip をダウンロードして、 解凍後に /Applications へ移動してください。
